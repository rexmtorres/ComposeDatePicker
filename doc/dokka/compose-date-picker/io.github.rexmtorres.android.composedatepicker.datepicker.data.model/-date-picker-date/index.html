DatePickerDate

@Stable
@Immutable
data class DatePickerDate(val year: Int, val month: Int, val day: Int)

Represents the date used by DatePicker.

There are several extension functions to convert between DatePickerDate and Calendar, Date and Long (date in milliseconds). Please refer to the following extensions:

  • Calendar.toDatePickerDate

  • Date.toDatePickerDate

  • Long.millisToDatePickerDate

  • DatePickerDate.toCalendar

  • DatePickerDate.toDate

  • DatePickerDate.toMillis

There are also extension functions to add days, months and years to a DatePickerDate. Please see DatePickerDate.addDays, DatePickerDate.addMonths, DatePickerDate.addYears.
