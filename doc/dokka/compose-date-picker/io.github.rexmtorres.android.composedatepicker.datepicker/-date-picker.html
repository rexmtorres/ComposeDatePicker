DatePicker

@Composable
fun DatePicker(modifier: Modifier = Modifier, date: DatePickerDate = DatePickerDate.currentDate, selectionLimiter: SelectionLimiter = SelectionLimiter.unlimited(), configuration: DatePickerConfiguration = DatePickerConfiguration.Builder().build(), id: Int = 1, locale: Locale = Locale.getDefault(), onMonthPageChange: (firstDayOfMonth: DatePickerDate, lastDayOfMonth: DatePickerDate) -> Unit = { _, _ -> }, onDateSelected: (DatePickerDate) -> Unit)

Composes a date picker.

Parameters

modifier

Modifier to be applied to the date picker.

date

The initial date.

selectionLimiter

SelectionLimiter that indicates the range of selectable dates.

configuration

DatePickerConfiguration that controls the appearance of the date picker.

id

Unique identifier for the date picker. This is important when displaying multiple date pickers in the same screen.

locale

The locale to use for displaying the month and day names.

onMonthPageChange

Callback invoked when the month changes as a result of clicking on the previous/next month arrows or selecting the month/year from the header.

onDateSelected

Callback invoked when a date is selected.
